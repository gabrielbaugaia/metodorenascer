

# Duas novas funcionalidades: Upload em lote de fitness + Anotações por treino

## Funcionalidade 1: Upload em lote de imagens de fitness (até 7)

### Problema
Hoje o limite é 3 imagens por dia. Se o aluno esquece de registrar por vários dias, precisa registrar dia a dia manualmente. Não há opção de enviar múltiplas imagens de uma vez para cobrir a semana.

### Solução
Criar uma nova seção/modal "Recuperar Semana" acessível na página Hoje (Renascer) que permita:
- Upload de até 7 imagens de uma vez (1 por dia da semana)
- A IA lê cada imagem via `extract-fitness
